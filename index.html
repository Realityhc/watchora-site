iOS application

Watchora

```

A simple, local-first app for personally tracking anime, movies, and TV series in one library.

Planned features

  • One library for anime, movies, and TV series
  • Watch status and episode progress
  • Personal ratings and notes
  • Manual media entries
  • Searchable titles, metadata, and poster artwork
Currently in development
Watchora is planned as a free, non-commercial application with no advertisements, subscriptions, or in-app purchases. TMDb API integration is planned for media metadata and poster images but has not yet been enabled.
```
